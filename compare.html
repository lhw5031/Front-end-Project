Java ↔ Python 코드 변환기

Java 공식 문서 Python 공식 문서

구글 번역기처럼 한쪽에 코드를 입력하면 다른 언어의 비슷한 OOP 문법으로 변환해 보여줍니다. 클래스, 생성자, 상속, 출력문 같은 기본 패턴을 빠르게 비교할 수 있습니다.

현재 변환기는 학습용 규칙 기반 도구입니다. 복잡한 프로젝트 전체보다 짧은 문법 예제와 OOP 구조 비교에 적합합니다.

지원하는 변환 패턴

클래스와 상속

class Dog extends Animal과 class Dog(Animal) 형태를 비교합니다.

생성자

Java 생성자와 Python __init__ 구조를 서로 대응시킵니다.

출력문

System.out.println과 print를 변환합니다.

기본 문법

세미콜론, 중괄호, this, self 같은 차이를 정리합니다.

Syntax Compare

코드 문법 비교

클래스

Java

class User {
  String name;
  int level;
}

Python

class User:
    def __init__(self, name, level):
        self.name = name
        self.level = level

생성자

Java

class User {
  User(String name) {
    this.name = name;
  }
}

Python

class User:
    def __init__(self, name):
        self.name = name

상속

Java

class Dog extends Animal {
  void speak() {
    System.out.println("bark");
  }
}

Python

class Dog(Animal):
    def speak(self):
        print("bark")

예외 처리

Java

try {
  int result = 10 / number;
} catch (ArithmeticException e) {
  System.out.println("error");
}

Python

try:
    result = 10 / number
except ZeroDivisionError:
    print("error")
