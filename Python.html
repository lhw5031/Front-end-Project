Language / Python

Python

Python 공식 문서 Python 튜토리얼

Python은 읽기 쉬운 문법과 풍부한 표준 라이브러리를 가진 범용 프로그래밍 언어입니다. 웹 개발, 데이터 분석, 자동화, 인공지능, 교육용 프로그래밍에 널리 쓰이며 객체지향과 함수형 스타일을 모두 지원합니다.

변수

변수는 값을 가리키는 이름입니다. Python은 타입을 직접 적지 않아도 값에 따라 타입이 정해지는 동적 타입 언어입니다. 

age = 20
name = "Jin"
is_student = True

자료형

Python에는 숫자, 문자열, 불리언 같은 기본 자료형과 리스트, 튜플, 딕셔너리 같은 컬렉션 자료형이 있습니다.

기본 자료형

int, float, str, bool

컬렉션

list, tuple, dict, set

연산자

연산자는 값을 계산하거나 비교할 때 사용합니다. 산술, 비교, 논리, 멤버십 연산자가 자주 쓰입니다. 

total = 10 + 5
is_valid = total >= 10 and total < 20
has_python = "Python" in ["Java", "Python"]

조건문

조건문은 조건의 참과 거짓에 따라 다른 코드를 실행합니다. Python은 중괄호 대신 들여쓰기로 코드 블록을 구분합니다. 

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
else:
    grade = "C"

반복문

반복문은 같은 작업을 여러 번 수행합니다. Python에서는 순회할 대상이 있으면 for, 조건을 기준으로 반복하면 while을 사용합니다. 

for i in range(3):
    print(i)

리스트

리스트는 여러 값을 순서대로 저장하는 변경 가능한 컬렉션입니다. 값을 추가, 수정, 삭제할 수 있어 가장 자주 쓰이는 자료구조 중 하나입니다. 

names = ["Jin", "Mina"]
names.append("Soo")
print(names[0])

튜플

튜플은 순서가 있는 컬렉션이지만 한 번 만들면 값을 바꿀 수 없습니다. 좌표, 고정된 설정값처럼 변하지 않는 묶음을 표현할 때 좋습니다. 

point = (10, 20)
x, y = point

딕셔너리

딕셔너리는 키와 값을 한 쌍으로 저장합니다. 키를 이용해 원하는 값을 빠르게 찾을 수 있습니다. 

scores = {"Python": 95, "Java": 90}
scores["OOP"] = 100
print(scores["Python"])

함수

함수는 특정 작업을 수행하는 코드 묶음입니다. def 키워드로 만들고, return으로 결과를 돌려줄 수 있습니다. 

def add(a, b):
    return a + b

매개변수

매개변수는 함수가 외부에서 받는 입력값입니다. 기본값, 키워드 인자, 가변 인자를 사용해 유연하게 설계할 수 있습니다. 

def greet(name, message="Hello"):
    return f"{message}, {name}"

greet(name="Jin")

클래스

클래스는 객체를 만들기 위한 설계도입니다. 객체가 가진 상태는 속성으로, 할 수 있는 행동은 메서드로 표현합니다. 

class User:
    def say_hello(self):
        print("Hello")

객체

객체는 클래스로부터 만들어진 실제 값입니다. 클래스 이름을 함수처럼 호출해 생성하고, 점 표기법으로 속성과 메서드에 접근합니다. 

user = User()
user.say_hello()

생성자

Python의 생성자 역할은 __init__ 메서드가 담당합니다. 객체가 생성된 직후 호출되어 초기 속성을 준비합니다. 

class User:
    def __init__(self, name):
        self.name = name

self

self는 현재 객체 자신을 가리키는 관례적인 이름입니다. 인스턴스 메서드의 첫 번째 매개변수로 사용하며, 객체의 속성과 메서드에 접근합니다. 

def rename(self, name):
    self.name = name

캡슐화

캡슐화는 객체 내부 상태를 직접 노출하지 않고 메서드를 통해 다루는 방식입니다. Python에서는 언더스코어 관례와 프로퍼티를 활용합니다. 

class Account:
    def __init__(self):
        self._balance = 0

    def deposit(self, amount):
        if amount > 0:
            self._balance += amount

상속

상속은 부모 클래스의 속성과 메서드를 자식 클래스가 물려받는 기능입니다. Python에서는 부모 클래스를 괄호 안에 적습니다. 

class Animal:
    def speak(self):
        return "sound"

class Dog(Animal):
    pass

오버라이딩

오버라이딩은 부모 클래스의 메서드를 자식 클래스에서 다시 정의하는 것입니다. 같은 메서드 이름으로 자식 객체만의 동작을 만들 수 있습니다. 

class Dog(Animal):
    def speak(self):
        return "bark"

다형성

다형성은 서로 다른 객체가 같은 메서드 호출에 각자의 방식으로 응답하는 특징입니다. 

animals = [Dog(), Cat()]

for animal in animals:
    print(animal.speak())

추상 클래스

추상 클래스는 공통 구조를 정의하고, 자식 클래스가 반드시 구현해야 할 메서드를 강제할 때 사용합니다. Python에서는 abc 모듈을 사용합니다. 

from abc import ABC, abstractmethod

class Shape(ABC):
    @abstractmethod
    def area(self):
        pass

Duck Typing

Duck Typing은 객체의 실제 타입보다 필요한 메서드를 가지고 있는지가 더 중요하다는 Python다운 사고방식입니다. 

def make_sound(animal):
    print(animal.speak())

make_sound(Dog())

staticmethod

@staticmethod는 객체 상태나 클래스 상태에 의존하지 않는 메서드를 클래스 안에 묶어둘 때 사용합니다. 

class MathUtil:
    @staticmethod
    def square(n):
        return n * n

클래스 변수

클래스 변수는 모든 객체가 공유하는 값입니다. 인스턴스마다 다른 값은 self에 저장하고, 공통 값은 클래스에 저장합니다. 

class User:
    count = 0

    def __init__(self):
        User.count += 1

예외 처리

예외 처리는 실행 중 발생할 수 있는 오류를 안전하게 처리하는 방법입니다. try, except, finally를 사용합니다. 

try:
    result = 10 / number
except ZeroDivisionError:
    print("0으로 나눌 수 없습니다.")

파일 입출력

파일 입출력은 파일에서 데이터를 읽거나 파일에 데이터를 저장하는 기능입니다. with 문을 사용하면 파일을 안전하게 닫을 수 있습니다. 

with open("memo.txt", "w", encoding="utf-8") as file:
    file.write("Hello Python")

with open("memo.txt", "r", encoding="utf-8") as file:
    text = file.read()

모듈

모듈은 Python 코드가 들어 있는 하나의 파일입니다. 관련 함수와 클래스를 파일 단위로 나누어 관리하고 import로 불러옵니다. 

import math

print(math.sqrt(16))

패키지

패키지는 여러 모듈을 폴더 단위로 묶은 구조입니다. 규모가 커진 프로젝트에서 기능별로 코드를 정리할 때 사용합니다. 

from app.models.user import User

user = User("Jin")

리스트 컴프리헨션

리스트 컴프리헨션은 반복문과 조건문을 이용해 새 리스트를 간결하게 만드는 문법입니다. 

squares = [n * n for n in range(5)]
even_numbers = [n for n in range(10) if n % 2 == 0]

람다 함수

람다 함수는 이름 없이 짧게 작성하는 함수입니다. 간단한 변환이나 정렬 기준을 넘길 때 자주 사용합니다. 

numbers = [3, 1, 2]
sorted_numbers = sorted(numbers, key=lambda n: n)
